### 色彩の意味・用途

#### 🌱 若草色（Primary）
- **意味**: 新緑の優しさ、家族の成長、希望
- **用途**: メインボタン、ナビゲーション、重要なアクション

#### ☀️ 光の黄色（Secondary）
- **意味**: 暖かい日だまり、家族の温もり、安心感
- **用途**: 背景色、カード背景、温かみのある領域

#### 🌳 土の茶色（Accent）
- **意味**: 大地の安定感、落ち着き、信頼
- **用途**: フッター、落ち着いたアクセント、セパレーター

## 🎭 UI/UXコンセプト

### 設計思想の反映

#### 「焦点化」の徹底
- **シングルカラム**: 縦一列レイアウトで視線誘導
- **中央揃え**: 重要な要素は画面中央に配置
- **余白活用**: 適切な space-y-* で要素間隔を確保

#### 「世界観」の醸成
```css
/* 優しい言葉遣いの視覚化 */
.gentle-text {
  font-weight: 400-500; /* 強すぎない太さ */
  line-height: 1.6;     /* ゆとりのある行間 */
  letter-spacing: 0.025em; /* 読みやすい字間 */
}

/* 温かみのあるアニメーション */
.warm-interaction {
  transition: all 0.2s ease-in-out;
  hover:scale-[102%]; /* 控えめな拡大 */
  hover:opacity-90;   /* 柔らかい変化 */
}
```

### ドキュメント更新

#### 更新ルール
1. **新コンポーネント追加時**: このドキュメントにパターン追加
2. **色変更時**: カラーパレット仕様を更新  
3. **レイアウト変更時**: レイアウトパターンを更新
4. **ブレークポイント変更時**: レスポンシブ設計を更新

#### 承認フロー
- **小さな変更**: 開発者レビュー
- **大きな変更**: デザインレビュー + 承認
- **破壊的変更**: チーム全体での合意
