## 2. 背景・目的

プロジェクトのUI/UXの指針として、旧サイト `uchinokiroku.com` のシンプルで温かみのあるデザインを継承することが決定された。
本タスクの目的は、現在のプロジェクトのトップページ (`/`) を、この決定された指針に基づき、旧サイトのルック＆フィールを再現するように改修することである。

## 3. 作業内容

現在のトップページ (`src/app/page.tsx`) のコンポーネント構造とスタイリングを全面的に改修する。
改修にあたっては、必ず関連ドキュメント、特に `UIUX分析_uchinokiroku.com.md` を熟読し、その設計思想を正確に反映させること。

- **修正対象ファイル（主）:**
    - `src/app/page.tsx`
    - `src/app/globals.css`

### 3.1. レイアウトの変更

- **要件:** ページ上のすべての要素が、画面の水平・垂直方向に対して中央に配置されるようにする。
- **実装ヒント:** `<body>` またはページのルートとなる `<div>` に `display: flex`, `justify-content: center`, `align-items: center` を適用し、`min-height: 100vh` を設定することで実現できる。

### 3.2. コンポーネントの再構成

`src/app/page.tsx` の内容を、以下の構造に沿って書き換える。

1.  **サイトタイトル:** `<h1>` 要素で `🏠 うちのきろく` を表示する。
2.  **キャッチコピー:** `<p>` 要素で `家族のあたたかい思い出をつづる場所です 💝` を表示する。
3.  **認証フォーム:** `<form>` 要素を配置し、以下の要素を含む。
    - **ラベル:** `<label>` で `🔑 あいことばを教えてください` を表示する。
    - **入力欄:** `<input type="password">` を配置する。`placeholder` にも「あいことば」と表示することが望ましい。
    - **送信ボタン:** `<button type="submit">` で `🏠おうちに入る` を表示する。
4.  **補足説明:** フォームの下に、小さな文字で `ご家族から教えてもらったあいことばを入力してください` という案内を表示する。

### 3.3. スタイリングの調整

- **背景色:** `globals.css` の `body` スタイルを調整し、ページ全体の背景色を温かみのある色（例: `#f3eac2` のような光の黄色、またはそれに類する色）に変更する。
- **フォント:** `layout.tsx` または `globals.css` で、サイト全体の基本フォントを、丸みを帯びた優しい印象のゴシック体（例: Google Fontsの `M PLUS Rounded 1c` や `Noto Sans JP` など）に設定する。
- **フォーム要素:**
    - `input` と `button` は、角丸 (`rounded-lg`) で、シンプルなデザインにする。
    - ボタンには、控えめな `hover` エフェクト（例: `hover:opacity-80`）を設定する。
- **テキスト:** 全体的に、文字サイズや行間を調整し、ゆとりのある読みやすいデザインにする。

## 4. 検証方法

1.  ローカル環境で `http://localhost:3000` にアクセスする。
2.  表示されるページが、`uchinokiroku.com` のスクリーンショットや `UIUX分析` ドキュメントに記載されたレイアウト・雰囲気と酷似していることを確認する。
3.  画面中央にすべての要素が配置されていることを確認する。
4.  異なる画面幅（PC、スマートフォン）でもレイアウトが崩れず、中央揃えが維持されることを確認する（簡易的なレスポンシブチェック）。

以上で作業は完了です。不明点があれば、PMまで確認してください。
