# TSK-057: サイドメニュー「月別アーカイブ」の階層表示とトグル開閉機能実装

## 1. 目的
左サイドメニューの「月別アーカイブ」を年・月・記事タイトルの階層表示にし、トグル開閉機能と最新年・月デフォルト開閉ロジックを実装する。

## 2. 詳細
ユーザーからのフィードバックによると、左サイドメニューの「月別アーカイブ」の表示が単調であり、記事の整理や発見がしにくい。年、月、記事タイトルの階層構造で表示し、トグルで開閉できるようにすることで、ナビゲーションの利便性を向上させる。また、最新の年と月はデフォルトで展開されているようにする。

## 3. 調査・修正項目
- 月別アーカイブのデータ構造を確認する。必要であれば、年、月、記事タイトルを含む階層的なデータを取得するAPIまたはロジックを検討する。
- サイドバーコンポーネント（`src/components/Sidebar.tsx`）の「月別アーカイブ」セクションの構造を確認する。
- 年、月、記事タイトルをネストしたUIコンポーネントを実装する。
- 各年、各月の項目にトグル開閉機能を追加する。開閉状態を示すアイコン（例: 矢印アイコン）も追加する。
- 最新の年と月は、ページロード時にデフォルトで展開されているようにロジックを実装する。
- 各記事タイトルがクリック可能で、記事詳細ページへ遷移するようにリンクを設定する。

## 4. 関連情報
- サイドバーコンポーネント: `src/components/Sidebar.tsx`
- 月別アーカイブのデータ取得ロジック（もしあれば）
- 記事詳細ページへのリンク構造
