# タスク詳細: TSK-010

**ID**: `TSK-010`
**タイトル**: 基本レイアウト（サイドバーとメインコンテンツ）の構築
**ステータス**: 未着手
**優先度**: 高

## 1. タスクの目的

PC版の左サイドバーとメインコンテンツエリアの基本構造を実装する。モバイル版のハンバーガーメニューの骨格も含む。これにより、アプリケーションの主要な画面構成を確立する。

## 2. 手順

以下の点を考慮し、Next.jsのApp Routerの仕組みを利用して、共通レイアウトを実装してください。

*   **PC版**: 画面左側に固定のサイドバー、右側にメインコンテンツエリアを配置する。
*   **モバイル版**: 左サイドバーはハンバーガーメニューに格納される構造とする。
*   **旧プロジェクトのUI/UXを尊重**: 旧プロジェクト(`uchi`)のUI/UX関連ファイル（例: `tailwind.config.js`, `globals.css`, `src/app/layout.tsx`など）を参考に、デザインや構造を継承してください。

## 3. 完了の定義

*   アプリケーションの共通レイアウトが実装され、PC版とモバイル版の基本的な表示が確認できること。

## 4. 検証方法

PMがコードを読み取り、レイアウト構造が意図通りに実装されていることを確認する。
