# TSK-054: 記事投稿ページへの画像アップロード機能実装

## 1. 目的
記事投稿ページから画像をローカルからアップロードできる機能を追加する。

## 2. 詳細
ユーザーからのフィードバックによると、記事投稿ページに画像アップロード機能がない。記事に画像を挿入できるようにするため、ローカル環境から画像をアップロードし、記事に埋め込む機能が必要。

TSK-023「画像アップロードAPIの実装」は完了しているが、これはAPI側の実装であり、フロントエンドからのアップロードUIと連携ロジックは未実装である。

## 3. 調査・修正項目
- 記事投稿ページ（`/articles/new` および `/articles/[slug]/edit`）に画像アップロード用のUI要素（ボタン、ドラッグ＆ドロップエリアなど）を追加する。
- 画像選択後、TSK-023で実装された画像アップロードAPI（`/api/upload` など）を呼び出し、画像をサーバーに送信するロジックを実装する。
- アップロード成功後、返却された画像URLを記事の本文（マークダウン）に挿入する機能（例: `![alt text](image_url)` 形式）を実装する。
- アップロード中のローディング表示、エラー時のフィードバックなどを実装する。
- アップロードされた画像の表示確認。

## 4. 関連情報
- TSK-023: 画像アップロードAPIの実装
- 記事投稿ページ: `src/app/articles/new/page.tsx`
- 記事編集ページ: `src/app/articles/[slug]/edit/page.tsx`
- 画像アップロードAPI: `src/app/api/upload/route.ts` (仮定)
