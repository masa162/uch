# TSK-056: サイドメニュー「タグ一覧」のトグル開閉機能実装

## 1. 目的
左サイドメニューの「タグ一覧」にトグル開閉機能を追加する。

## 2. 詳細
ユーザーからのフィードバックによると、左サイドメニューの「タグ一覧」が常に展開されており、表示領域を占有している。タグの数が増えた場合にメニューが長くなりすぎることを防ぐため、トグルで開閉できるようにする。

## 3. 調査・修正項目
- サイドバーコンポーネント（`src/components/Sidebar.tsx`）の「タグ一覧」セクションの構造を確認する。
- タグ一覧の表示/非表示を切り替えるための状態（state）をコンポーネントに追加する。
- タグ一覧のタイトル部分（例: 「タグ一覧」というテキスト）をクリックすることで、状態が切り替わり、タグリストが表示/非表示になるように実装する。
- 開閉状態を示すアイコン（例: 矢印アイコン）を追加し、状態に応じてアイコンの向きが変わるようにする。

## 4. 関連情報
- サイドバーコンポーネント: `src/components/Sidebar.tsx`
- タグ一覧のデータ取得ロジック
