# TSK-048: OGP画像の表示とTOP回帰リンクの設定

## 1. 目的
左上OGP画像が表示されない問題を修正し、TOPページへの回帰リンクとして機能させる。

## 2. 詳細
ユーザーからの報告によると、Webサイトの左上にOGP画像が表示されていない。この画像はサイトのアイデンティティを示す重要な要素であり、クリックすることでTOPページへ遷移するリンクとしても機能させる必要がある。

TSK-009「OGP画像・ファビコンの配置」でOGP画像は配置済みのはずだが、表示されていない原因を調査し、修正する。

## 3. 調査・修正項目
- OGP画像が正しく配置されているか、パスが正しいか確認する。
- CSSやHTMLの記述に問題がないか確認する。
- 画像が表示されない原因（例: ブロックされている、パスが間違っている、CSSで非表示になっているなど）を特定する。
- OGP画像をTOPページへのリンクとして設定する。
