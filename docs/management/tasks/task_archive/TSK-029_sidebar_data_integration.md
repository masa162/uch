# タスク詳細: TSK-029

**ID**: `TSK-029`
**タイトル**: サイドバーのデータ連携
**ステータス**: 未着手
**優先度**: 高

## 1. タスクの目的

サイドバーのタグ一覧、月別アーカイブを、記事発見APIと連携させ、動的なデータを表示する。これにより、サイドバーが実際のデータに基づいて機能するようになる。

## 2. 手順

### 2.1. `src/components/Sidebar.tsx` の修正

*   **目的**: タグ一覧取得API (`/api/articles/tags`) と月別アーカイブ一覧取得API (`/api/articles/archive`) を呼び出し、サイドバーに動的なデータを表示する。
*   **修正箇所**: `Sidebar.tsx` 内でAPIを呼び出し、取得したデータを表示するロジックを追加する。

## 3. 完了の定義

*   サイドバーのタグ一覧がAPIから取得したデータに基づいて表示されること。
*   サイドバーの月別アーカイブがAPIから取得したデータに基づいて表示されること。

## 4. 検証方法

PMがコードを読み取り、API連携ロジックが意図通りに実装されていることを確認する。
