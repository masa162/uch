# タスク詳細: TSK-033

**ID**: `TSK-033`
**タイトル**: 環境クリーンアップと不要資産の棚卸し
**ステータス**: 未着手
**優先度**: 高
**担当者**: Gemini CLI PM / Claude Code

## 1. タスクの目的

プロジェクト環境（ローカル、Docker、VPS）から不要なファイル、設定、リソースを特定し、クリーンアップすることで、環境の一貫性と健全性を確保する。

## 2. 背景

旧リポジトリからの移行に伴い、各環境に「遺産」として不要なファイルや設定が残存している可能性がある。これらは将来的な問題や混乱の原因となるため、早期に排除する必要がある。

## 3. 具体的な作業内容

*   **ローカル環境**:
    *   旧プロジェクト関連のファイルやディレクトリの特定と削除。
    *   不要な依存関係（`package.json`、`package-lock.json`）の確認とクリーンアップ。
    *   `.env` ファイルなどの機密情報の適切な管理。
*   **Docker環境**:
    *   未使用のDockerイメージ、コンテナ、ボリューム、ネットワークの特定と削除。
    *   `docker-compose.yml` や `Dockerfile` 内の不要な記述の確認と削除。
*   **VPS環境**:
    *   旧プロジェクト関連のファイルやディレクトリの特定と削除（特に `/root` や `/var/www` など）。
    *   未使用のポートの特定と閉鎖（ファイアウォール設定の確認）。
    *   古いDockerイメージ、コンテナ、ボリューム、ネットワークの特定と削除。
    *   VPSのOSレベルでの不要なパッケージやサービスの確認と削除。

## 4. 完了の定義

*   各環境における不要なリソースが特定され、クリーンアップ計画が策定される。
*   クリーンアップが実行され、環境が整理される。
*   クリーンアップ結果が文書化される。
