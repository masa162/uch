# タスク詳細: TSK-011

**ID**: `TSK-011`
**タイトル**: サイドバー共通コンポーネントの実装
**ステータス**: 未着手
**優先度**: 高

## 1. タスクの目的

左サイドバー内に表示される各要素（OGP画像、検索ボックス、タグ一覧、月別アーカイブ、エッセイ項目）のコンポーネントを実装する。これにより、ナビゲーションと情報表示の基盤を構築する。

## 2. 手順

以下のコンポーネントを実装し、`TSK-010`で構築した基本レイアウトのサイドバーに組み込んでください。

*   **OGP画像**: 「うちのきろく」ロゴ兼トップページリンクとして機能する。
*   **検索ボックス**: 検索機能のUI部分。
*   **タグ一覧**: トグル開閉可能で、タグをリスト表示する。
*   **月別アーカイブ**: トグル開閉可能で、最新年/月がデフォルトで展開される。月別リンクを表示する。
*   **エッセイ項目**: 固定のリンクまたはテキスト表示。

*   **旧プロジェクトのUI/UXを尊重**: 旧プロジェクト(`uchi`)のUI/UX関連ファイル（例: `src/components/sidebar.tsx`など）を参考に、デザインや構造を継承してください。

## 3. 完了の定義

*   サイドバー内の各コンポーネントが実装され、意図通りに表示されること。
*   トグル開閉機能が動作すること。

## 4. 検証方法

PMがコードを読み取り、コンポーネントの実装と機能が意図通りであることを確認する。
