# TSK-052: 家族ローンチ前の認証・あいことば設定

## 1. 目的
家族ローンチ前に、本番環境でのあいことば/認証設定を有効にする。

## 2. 詳細
ユーザーからの指示により、現段階では開発・デバッグの便宜のため、本番環境での認証はスキップされている。しかし、家族ローンチ（一般公開）前には、セキュリティとプライバシー保護のため、あいことばまたは適切な認証設定を有効にする必要がある。

## 3. 実施項目
- TSK-047で確認・修正された認証スキップ設定を無効化する。
- `NEXT_PUBLIC_SITE_PASSWORD` 環境変数を設定し、サイトパスワード認証を有効にする。
- 必要に応じて、OAuth認証（Google, LINE）が正しく機能することを確認する。
- 認証が有効になった状態で、主要なページへのアクセス、記事の閲覧、投稿などが正しく行えるかテストする。
