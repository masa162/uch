# TSK-055: 記事一覧カードのリンク領域拡大

## 1. 目的
記事一覧ページの記事カード全体にリンク領域を拡大し、UI/UXを改善する。

## 2. 詳細
ユーザーからのフィードバックによると、記事一覧ページ（`/articles`）に表示される記事カードにおいて、「詳細をみる」ボタンのみがリンクになっており、クリック領域が小さく操作性が悪い。カード全体をクリック可能にすることで、ユーザーの利便性を向上させる。

## 3. 調査・修正項目
- 記事一覧ページ（`/articles/page.tsx`）のカードコンポーネントの構造を確認する。
- 現在「詳細をみる」ボタンに設定されているリンクを、カード全体をラップする要素（例: `Link` コンポーネントや `<a>` タグ）に移動させる。
- カード全体がクリック可能になった際に、視覚的にリンクであることがわかるようなホバーエフェクトなどを追加する。
- 既存の「詳細をみる」ボタンは削除するか、デザインを調整する。

## 4. 関連情報
- 記事一覧ページ: `src/app/articles/page.tsx`
- 記事カードのコンポーネント（もしあれば）
